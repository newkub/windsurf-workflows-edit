# Windsurf Workflows Editor

This project is a web-based editor for managing Windsurf workflow files (`.md`). It allows users to view a list of available workflows and edit their content directly in the browser.

## Tech Stack

- [Nuxt](https://nuxt.com/)
- [Vue](https://vuejs.org/)
- [UnoCSS](https://unocss.dev/)
- [Biome](https://biomejs.dev/) for linting and formatting
- [Vitest](https://vitest.dev/) for testing

## Setup

Install the dependencies using Bun:

```bash
bun install
```

## Available Scripts

### `bun run dev`

Starts the development server on `http://localhost:3000`.

### `bun run build`

Builds the application for production.

### `bun run lint`

Lints and formats the codebase using Biome.

### `bunx vitest run`

Runs the test suite.
